# AI 開發教學網站

> 分享使用 AI 進行軟體開發的實際經驗與技巧

## 專案簡介

這是一個新手友善的 AI 開發指南，透過實戰經驗分享如何有效運用 AI 工具提升開發效率。

## 目的

- 幫助開發者快速上手 AI 輔助開發
- 分享實際可行的工作流程和技巧
- 建立以人為中心的 AI 開發方法論

## 如何使用

直接打開 `index.html` 即可瀏覽完整內容。

## 技術棧

- HTML5 + CSS3 + Vanilla JavaScript
- 無框架依賴，純原生實現
- 適合任何瀏覽器直接開啟

## 內容章節

1. AI 開發基本心態
2. 工具選擇與設定
3. Prompt 實戰技巧
4. 除錯與疑難排解
5. 專案管理工作流程
6. 進階應用與自動化
7. 踩坑經驗分享
8. 學習資源與社群
