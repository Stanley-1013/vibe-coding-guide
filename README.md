# AI 開發教學網站

精美、流暢的教學小網站，分享使用 AI 進行軟體開發的實際經驗與技巧。

## 🌐 線上預覽

訪問：[https://stanley-1013.github.io/vibe-coding-guide/](https://stanley-1013.github.io/vibe-coding-guide/)

## 📚 特色

- 🎯 新手友善：術語最少，直接可用的方法
- 📋 可複製 Prompt：每個技巧都有實用的 AI prompt
- 💡 真實經驗：第一人稱分享實際使用心得
- 📱 響應式設計：支援桌面、平板、手機

## 🛠️ 技術棧

- HTML5 + CSS3 + Vanilla JavaScript
- 無框架依賴，純靜態網站
- GitHub Pages 託管

## 📖 內容結構

- **工具入口**: Antigravity、Claude Code、其他工具
- **核心技巧**: 3 個立即有效的技巧
- **進階方式**: Skill / Rule / MCP 概念
- **FAQ**: 5 個常見問題解答
- **小字典**: 必要術語一句話說明
- **彩蛋**: 本站開發過程展示

## 🚀 本地開發

```bash
# 直接打開
open index.html

# 或使用本地伺服器
python3 -m http.server 8000
# 訪問 http://localhost:8000
```
